HTML: HyperText Markup Language

HyperText는 Text들이 서로 연결됨을 뜻하는 용어입니다.
Text들이 서로 떨어져 있어도 독자들은 그 사이를 즉시 이동할 수 있는 것이죠.
인터넷이 만들어지기도 한참 전인 1960년대에 만들어진 용어라고 합니다.
어쨌든, 당연히 HTML엔 웹 페이지를 연결해주는 기능이 있고,
그것을 hyperlink, 또는 줄여서 link라고 부릅니다. 그것을 수행해 주는 태그를 알아봅시다.
<a> </a>를 사용하면 되는데요, a는 anchor의 약자입니다.
이 태그를 닻으로 삼아 다른 페이지로 연결된다는 의미에요!
태그 사이에 오는 텍스트에 링크가 걸릴 건데요, 그럼 그 링크가 어디로 연결될 것인지는
href를 통해 지정합니다. href는 hyper reference입니다!
<a href="https://boxtobox.github.io/htmlcsspractice/css.html"> CSS </a>
이렇게요!
마지막으로, 이렇게 만든 링크를 클릭하면 같은 창에서 열리게 되는데 새 창에서 열리게 지정하고 싶다면,
<a target="_blank" href="https://boxtobox.github.io/htmlcsspractice/2-css.html"> CSS </a>
이렇게 해주면 됩니다! 아주 쉽네요.

CSS
참고로, href안에 저렇게 좀 전에 만든 페이지 주소(=url)를 넣어 주었는데요, 위 주소에선 이것들을 명시(specify)해주고 있어요!

  • https:// 어떤 프로토콜을 사용할지 지정했고요
  • boxtobox.github.io: 도메인도 알려줬고요.
  • /htmlcsspractice/2-CSS.html: 마지막으로 path(주소)까지 알려줬죠.
이렇게 브라우저가 필요로 하는 모든 정보를 제공하는 url를 absolute url이라고 부른다고 합니다.
반대로, 어떤 주소는 그냥 path만 가지고 있는데 그 경우엔 relative url이라고 합니다.
칸 아카데미 내부의 링크들은 relative url을 쓴다고 합니다.
a 태그 내에는 p태그도 들어갈 수 있고, img 태그도 들어갈 수 있어요. img 태그를 사용한 경우에는 이미지에 링크가 걸리죠!
