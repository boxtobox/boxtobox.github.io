CSS에 대해 알아보기

CSS는 Cascading Style Sheets의 약자로, html과 완전히 다른 문법을 사용하는 다른 언어라고 합니다. 목적은 페이지의 폰트, 색상 등 스타일을 바꿔주기 위함이죠
head 태그 안에 <style> </style> 태그를 넣어주면, style태그 안의 내용은 html 파서가 아닌 css 파서로 브라우저가 인식하게 됩니다.
우선은 h1의 색상을 바꿔주었어요.
h1 { color:rgb(0, 47, 255);}
이제 이 문서에서 사용하는 모든 h1 태그를 이용한 제목의 스타일이(여기서는 색깔) 바뀌게 될 겁니다!
h1 뒤에 중괄호 그리고 color, 그리고 rgb 색상 코드가 옵니다.

태그 이름으로 선택하기

배경색과 본문 글자 색도 바꿨습니다.
참고로 p 태그는 paragraph의 줄임이에요.
<style>
       h1 { color:rgb(0, 47, 255);}
       p { color:rgb(189, 33, 33);}
       body { background-color: rgb(231, 227, 171);}
<style>

ID로 선택하기

style 태그 내에 적어놓은 h1, p, body를 selector라고 해요.
근데 위와 같이 style 태그를 적어놓으면, body 태그 내에 모든 h1, p가 바뀌게 될거에요.
예를 들어, 특정 p 태그에만 다른 스타일을 적용하고 싶다면,
ID를 이용해서 개별적인 스타일 적용이 가능합니다.

ID를 만들고 생성하기 위해, style 태그 내에 ID를 생성해 주고 property도 지정해 줍시다.
ID는 id-notes라고 할거고, 이름 앞에 샾(#)을 붙여줘야 해요.
#id-notes {color:darkorange; background-color:darkviolet;}
그리고, 이제 body에 삽입한 태그 중 하나에 이 ID를 적용합니다.

<p id="id-notes">
이런식으로요!
이 문단만 id-notes라는 ID를 할당받아서, 오렌지색의 글자와 보라색 배경을 갖게 됩니다.
</p>

ID 만들 때 주의할 점!

  • ID는 unique해야 합니다. body 내에 같은 ID를 둘 이상 사용할 수도 없어요.
    stlye 내에 ID를 생성한 selector 한개는 body 태그 내에 사용된 ID 한개와 일대일 대응만 가능하단거죠.
  • ID내에 공백은 허용되지 않아요. (Whitespaces are not allowed)
  • ID는 case sensitive합니다!

class로 선택하기

위에 첫번째 주의점에서 말했듯이, ID는 unique해야 해서 만약 여러개의 문단(p)에 같은 스타일을 적용하고 싶다면,
동일한 ID를 여러개 만들어야 하는데 정말 낭비인거에요!
이 문제를 해결할 수 있는 방법으로, class를 사용합니다.
class를 만드는 법은 ID 만드는 법과 같은데, 이름 앞에 #이 아닌 .을 사용합니다.

<style>
.brexit {
  color:rgb(50, 18, 233); background-color:rgb(169, 224, 199);
}
<style>

이제 이 스타일을 여러 문단에 적용해 볼게요!

2012년 데이비드 캐머런 영국 총리는 유럽연합 탈퇴를 묻는 국민투표 요구를 거부하였으나, 국민들의 요구가 있다면 가까운 미래에 실시 할 수 있다고 제안하였다.[12][13] BBC에 따르면, 캐머런 총리는 유럽연합 회원국으로서의 영국의 위치를 확고히 하고자하는 필요성은 영국 국민들의 깊은 가슴속 지지를 받고 있지만 전술적이고 전략적인 인내심이 요구된다고 하였다.[14]

2013년 1월 캐머런 총리는 2015년 총선에서 보수당이 집권한다면, 재교섭안으로서 2017년 말까지 유럽연합 탈퇴 여부를 묻는 주민투표를 실시할 것이라고 발표하였다.[15] 보수당은 2015년 영국 총선에서 승리하였고, 곧 영국 국회엔 유럽연합 주민투표 법안이 발의되었다. 당내의 양심투표 요구가 강해짐에 따라 캐머런 총리 본인은 개혁 약속을 받아낸 후 유럽 연합 잔류를 지지하였음에도, 내각 관료와 보수당 내 국회의원들에게 양심에 따라 찬반캠페인에 자유롭게 참여해도 된다고 허락하였다.[16][17][18]

2016년 2월 22일 캐머런 총리는 영국 하원에서 국민투표가 2016년 6월 23일에 열릴 것이며, 국민투표에 따라 영국의 유럽연합 탈퇴가 결정될 수 있을 것에 대비해 리스본 조약의 제 50조에 따른 법적 검토를 시작하였다고 발표했다.[19][20] 캐머런 총리는 리스본 조약에 따라 국민투표에서 유럽연합 탈퇴가 결정되면 개표결과가 발표되는 바로 그 순간부터 2년간의 탈퇴 협상이 시작된다고 말했다.[21]
